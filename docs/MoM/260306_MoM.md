# 1. 프로젝트 주제

## 흑백로그
음식점 리뷰 및 랭킹 시스템을 기반으로 한 서비스  

사용자가 음식점을 방문하고 **리뷰 및 도장을 기록하며 랭킹을 확인할 수 있는 플랫폼**

---

# 2. 주요 기능

## 2-1. 사용자 기능
- 음식점 리뷰 작성
- 음식점 방문 기록 및 도장 시스템
- 음식점 및 사용자 랭킹 확인
- 음식점 위치 지도 확인

## 2-2. 관리자 기능
- 음식점 등록

- 음식점 삭제
