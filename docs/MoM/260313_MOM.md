# [2026.03.13] 팀 회의록 (6차)

## 3줄 요약

- Git merge 이후 메인, 로그인, 마이페이지, owner/관리자 페이지 등 주요 페이지 연결 및 충돌 해결이 완료되었다.

- 현재 회원가입 페이지 merge를 진행 중이며, 파일 경로를 섹션별·유형별로 정리하는 구조 개편이 추가 안건으로 논의되었다.

- users 테이블 컬럼 2개가 추가되었고, 판매자 등록 허가 페이지 및 로그인/판매자 상태별 햄버거바 메뉴 분기 규칙이 새로 정리되었다.

## 1. 참여자

- 정유영
- 강인
- 민규동
- 이종민
- 장성원
- 조혜성

---

## 2. 완료 사항

- Git merge 이후 페이지 연결 작업 진행
- merge 해결 완료 페이지
  - 메인페이지
  - 지도 API
  - 로그인
  - 로그인 API
  - 마이페이지
  - 판매자 등록 신청 페이지
  - owner 관련 페이지 (board, menu, review, notice)
  - 관리자 페이지

---

## 3. 진행 중

- 회원가입 페이지 merge 진행 중

---

## 4. 추가 안건

- 파일 경로를 섹션별로 구분하여 저장하는 구조로 정리
- 예시 구조
  - 공통 CSS: `static/css`
  - 섹션별 데이터:
    - `static/pages/user`
    - `static/pages/owner`
- 파일 유형별 정리 기준 수립 필요
  - Python
  - CSS
  - HTML

---

## 5. 변동사항

- `users` 테이블 컬럼 2개 추가

---

## 6. 신규 논의 사항

- 판매자 등록 허가 페이지 구현
- 메인페이지 햄버거바 네비게이션바 설정

### 메뉴 표시 규칙

#### 1) 로그인 전
- 로그인
- 회원가입
- 공지사항
- 고객센터

#### 2) 로그인 후, 아직 판매자 등록 안 함
- 마이페이지
- 로그아웃
- 판매자 등록하기
- 공지사항
- 고객센터

#### 3) 로그인 후, 판매자 등록 완료
- 마이페이지
- 로그아웃
- 판매자 페이지
- 공지사항
- 고객센터
